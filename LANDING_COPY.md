# RelayKit Landing Copy

## Hero

Harness augmentation for multi-tool, human-in-the-loop parallel execution.

RelayKit augments Codex, Claude Code, Gemini CLI, and Antigravity so one operator can run parallel lanes, assign deliberate roles, and keep the human in charge of coordination.

## Subhead

Task intake, lane recommendation, harness onboarding, checkpoints, and prompt-stack resolution for teams that want multiple AI harnesses to behave like one operator-directed system.

## Primary Value Points

- Keep multiple harnesses active without losing control of scope or lane ownership.
- Start from a task, then get the smallest useful setup instead of hard-binding work to one tool.
- Let the harness own onboarding and wiring through MCP instead of making the user edit config by hand.
- Preserve a file-first runtime with inspectable task state, profiles, and prompt assets.

## Short Blurb

RelayKit is a harness-augmentation product for operators who want multiple AI harnesses to work in parallel without turning them into a black box swarm.
